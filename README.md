# 🚀 Event Booking MERN App

A full-stack **Event Booking Platform** built using the MERN stack (MongoDB, Express.js, React.js, Node.js). This application allows users to explore events, book tickets, and manage bookings with a secure authentication system.

---

## 🌟 Features

### 👤 User Features

* 🔐 JWT Authentication (Login/Register)
* 🎟️ Browse and book events
* 📩 Email OTP verification
* 📱 Responsive UI (Mobile + Desktop)
* 🧾 Booking history

### 🛠️ Admin Features

* ➕ Add / Edit / Delete Events
* 👥 Manage users
* 📊 Dashboard overview

---

## 🧑‍💻 Tech Stack

**Frontend:**

* React.js
* Axios
* React Router
* Tailwind CSS / CSS

**Backend:**

* Node.js
* Express.js
* MongoDB (Mongoose)
* JWT Authentication
* Nodemailer (for OTP)

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/amaanpasha123/Event-Booking.git
cd Event-Booking
```

---

### 2️⃣ Setup Backend

```
cd server
npm install
```

Create `.env` file:

```
MONGO_URI=your_mongodb_url
JWT_SECRET=your_secret_key
EMAIL_USER=your_email
EMAIL_PASS=your_email_password
```

Run backend:

```
npm start
```

---

### 3️⃣ Setup Frontend

```
cd client
npm install
npm run dev
```
